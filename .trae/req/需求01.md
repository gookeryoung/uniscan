# 需求01：基础功能

- [x] 1. 设计一款通用文件扫描器，使用 python+pyside2 技术栈，包括 cli 和 gui 界面。
- [x] 2. 使用 yaml 进行扫描规则配置，支持文件名、文件内容按特定字符匹配、RE 匹配等不同模式，并且支持简单的 AND、OR、NOT 等逻辑组合。
- [x] 3. 支持 PDF、DOCX、PPTX、EXCEL、ODT、WPS 等多种文件格式扫描。
- [x] 4. 支持 ZIP、RAR 压缩文件扫描。
- [x] 5. 支持长期托盘驻守，配置忽略目录（可提前配置如 WINDOWS 系统文件等），然后根据新增文件进行动态扫描，以提高效率。
