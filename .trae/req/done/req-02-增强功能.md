# 需求02：增强功能

- [x] 1. 扫描过程需要实现开始/暂停/继续（同一按钮）、停止等功能。
- [x] 2. 扫描过程中，需要实时显示扫描进度、扫描文件数量、扫描时间等信息。
- [x] 3. 扫描完成后，需要提供扫描结果的导出功能，支持导出为 CSV、JSON 等格式。
- [x] 4. 扫描结果需要支持根据扫描规则进行筛选，只显示符合规则的文件。
- [x] 5. 扫描结果需要支持根据扫描时间、扫描文件数量等进行排序。
- [x] 6. 扫描结果需要支持根据扫描文件路径进行筛选，只显示符合规则的文件。
- [x] 7. 扫描结果预览能够定位到具体命中位置，方便用户查看。
- [x] 8. 对于一个文件中存在多个命中记录的情况，需要提供方便的前后跳转功能。
- [x] 9. 使用 ui 设计界面。
- [x] 10. 增加修改规则文件功能。
- [x] 11. 启动时如果扫描路径有效，则启用扫描按钮，而不是非要选择才启用。
- [x] 12. 结果显示除了列表显示，还需要提供按规则、按严重等级等不同的分类显示规则，并且在标题部分提供排序操作功能。
